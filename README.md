# 📌 Project Overview

Smart City Traffic Forecasting is a Machine Learning application that predicts traffic volume at different city junctions using historical traffic data.

The project includes a complete Machine Learning pipeline with:

- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Traffic Prediction
- Interactive Streamlit Dashboard

---

# 🎯 Objectives

- Predict future traffic volume
- Help Smart City traffic management
- Visualize traffic patterns
- Compare model performance
- Build a production-style ML pipeline

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-City-Traffic-Forecasting.git
```

Move into project directory

```bash
cd Smart-City-Traffic-Forecasting
```

Create Virtual Environment

Windows

```bash
python -m venv smartcityenv
```

Activate

```bash
smartcityenv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🚀 Run Project

## Streamlit Dashboard

```bash
streamlit run app.py
```

## Command Line Version

```bash
python main.py
```
