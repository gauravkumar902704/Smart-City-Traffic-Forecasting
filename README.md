# рҹ“Ң Project Overview

Smart City Traffic Forecasting is a Machine Learning application that predicts traffic volume at different city junctions using historical traffic data.

The project includes a complete Machine Learning pipeline with:

- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Traffic Prediction
- Interactive Streamlit Dashboard

# вҡҷ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-City-Traffic-Forecasting.git
```

Move into project directory

```bash
cd Smart-City-Traffic-Forecasting
```

Create Virtual Environment

Windows

```bash
python -m venv smartcityenv
```

Activate

```bash
smartcityenv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

---

# рҹҡҖ Run Project

## Streamlit Dashboard
